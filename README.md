# 📝 NoteVerse

**NoteVerse** is a modern, feature-rich note-taking application built with **Next.js**, **Firebase**, and **Tailwind CSS**, designed for productivity, creativity, and security. Whether you're jotting down ideas, writing essays, or organizing tasks, NoteVerse is your intelligent, encrypted workspace.

---

## 🚀 Features

### 🔐 Authentication
- Google Sign-in
- Email/Password authentication
- Password reset support
- Secure session management

### 🗒️ Note Management
- Create, edit, and delete notes
- Rich text editing with formatting
- Pin important notes
- Color-code notes for organization
- Real-time sync via Firebase

### 🌟 Advanced Features
- 🔒 **End-to-End Encryption** for sensitive notes
- 🤖 **AI Writing Assistant** powered by **Google Gemini**:
  - Text summarization
  - Writing enhancement suggestions
  - Word definitions & contextual explanations

### 💡 User Experience
- 🌓 Dark & Light mode support
- 📱 Responsive design for all screen sizes
- 🔍 Full-text search across all notes
- 💨 Smooth transitions & animations
- 🎨 Customizable note colors
